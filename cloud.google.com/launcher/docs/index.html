What is Google Cloud Launcher?

Google Cloud Launcher lets you quickly deploy functional software packages that run on Google Cloud Platform. Even if you are unfamiliar with services like Compute Engine or Cloud Storage, you can easily start up a familiar software package without having to manually configure the software, virtual machine instances, storage, or network settings. Deploy a software package now, and scale that deployment later when your applications require additional capacity. Google Cloud Platform updates the images for these software packages to fix critical issues and vulnerabilities, but doesn't update software that you have already deployed.

Sign up for a free trial and get $300 in credits to create deployments of your favorite software packages.

Deploying a software package

You can select and deploy software packages from the Cloud Launcher page of the Cloud Platform Console. Cloud Launcher offers many different solutions, and in some cases, offers a few variations of the same solution; for example, Cloud Launcher has multiple packages for WordPress. To ensure a solution meets your needs, each solution has a details page that describes the VM type, operating system, estimated costs, and more.

Search for a package and select one that meets your business needs. When you launch the deployment, you can use the default configuration or customize the configuration to use more virtual CPUs or storage resources. Some packages allow you to specify the number of virtual machine instances to use in a cluster.

Billing

Most software packages in Cloud Launcher are free. The only costs to deploy these solutions are the normal usage fees for Google Cloud Platform resources. Estimated costs are based on the minimum recommended instance and storage configuration. The estimate does not include networking costs. You can modify the instance and storage configuration when you deploy the configuration.

If an image does incur a usage fee, the fee appears on your monthly Google Cloud Platform invoice as a separate line item. You are charged for images with usage fees the same way that you are charged for use of Google Cloud Platform services. For more information about Google Cloud Platform billing, refer to the Google Cloud Platform billing help.

Support

For most software packages on Cloud Launcher, you can find support contacts on the package details page for each package under the Maintenance and Support section.

You can also get support from the broader community on Stack Overflow.

Feedback and suggestions

You can suggest new software packages or provide feedback on the existing offerings through the feedback link at the bottom of the Cloud Launcher page.
