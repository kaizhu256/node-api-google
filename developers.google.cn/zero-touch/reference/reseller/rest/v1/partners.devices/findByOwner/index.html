Method: partners.devices.findByOwner

  • HTTP request
  • Path parameters
  • Request body
    • JSON representation
  • Response body
    • JSON representation
  • Authorization Scopes

Find devices by ownership.

HTTP request

POST https://androiddeviceprovisioning.googleapis.com/v1/partners/{partnerId}/devices:findByOwner

The URL uses Google API HTTP annotation syntax.

Path parameters

Parameters
partnerId

string (int64 format)

ID of the partner.

Request body

The request body contains data with the following structure:

JSON representation 
{
  "pageToken": string,
  "limit": string,
  "sectionType": enum(DeviceProvisioningSectionType),
  "customerId": [
    string
  ],
}
Fields
pageToken

string

Page token.

limit

string (int64 format)

The number of devices to show in the result.

sectionType

enum(DeviceProvisioningSectionType)

The section type.

customerId[]

string (int64 format)

List of customer IDs to search for.

Response body

If successful, the response body contains data with the following structure:

Response containing found devices.

JSON representation 
{
  "devices": [
    {
      object(Device)
    }
  ],
  "nextPageToken": string,
}
Fields
devices[]

object(Device)

Devices found.

nextPageToken

string

Page token of the next page.

Authorization Scopes

Requires the following OAuth scope:

  • https://www.googleapis.com/auth/androidworkprovisioning

For more information, see the Auth Guide.
