Resource: Creative

A creative and its classification data.

JSON representation 
{
  "accountId": string,
  "creativeId": string,
  "agencyId": string,
  "clickThroughUrls": [
    string
  ],
  "impressionTrackingUrls": [
    string
  ],
  "adChoicesDestinationUrl": string,
  "vendorIds": [
    number
  ],
  "attributes": [
    enum(CreativeAttribute)
  ],
  "restrictedCategories": [
    enum(RestrictedCategory)
  ],
  "advertiserName": string,
  "dealsStatus": enum(StatusType),
  "openAuctionStatus": enum(StatusType),
  "servingRestrictions": [
    {
      object(ServingRestriction)
    }
  ],
  "detectedAdvertiserIds": [
    string
  ],
  "detectedProductCategories": [
    number
  ],
  "detectedLanguages": [
    string
  ],
  "detectedDomains": [
    string
  ],
  "detectedSensitiveCategories": [
    number
  ],
  "corrections": [
    {
      object(Correction)
    }
  ],
  "filteringStats": {
    object(FilteringStats)
  },
  "version": number,
  "apiUpdateTime": string,

  // Union field content can be only one of the following:
  "html": {
    object(HtmlContent)
  },
  "video": {
    object(VideoContent)
  },
  "native": {
    object(NativeContent)
  },
  // End of list of possible types for union field content.
}
Fields
accountId

string

The account that this creative belongs to. Can be used to filter the response of the creatives.list method.

creativeId

string

The buyer-defined creative ID of this creative. Can be used to filter the response of the creatives.list method.

agencyId

string (int64 format)

The agency ID for this creative.

clickThroughUrls[]

string

The set of destination URLs for the creative.

impressionTrackingUrls[]

string

The set of URLs to be called to record an impression.

adChoicesDestinationUrl

string

The link to AdChoices destination page.

vendorIds[]

number

All vendor IDs for the ads that may be shown from this creative. See https://storage.googleapis.com/adx-rtb-dictionaries/vendors.txt for possible values.

attributes[]

enum(CreativeAttribute)

All attributes for the ads that may be shown from this creative. Can be used to filter the response of the creatives.list method.

restrictedCategories[]

enum(RestrictedCategory)

All restricted categories for the ads that may be shown from this creative.

advertiserName

string

The name of the company being advertised in the creative.

dealsStatus

enum(StatusType)

The top-level deals status of this creative. If disapproved, an entry for 'auctionType=DIRECT_DEALS' (or 'ALL') in servingRestrictions will also exist. Note that this may be nuanced with other contextual restrictions, in which case, it may be preferable to read from servingRestrictions directly. Can be used to filter the response of the creatives.list method.

Note: This field is used in responses only. Any value specified here in a request is ignored.
openAuctionStatus

enum(StatusType)

The top-level open auction status of this creative. If disapproved, an entry for 'auctionType = OPEN_AUCTION' (or 'ALL') in servingRestrictions will also exist. Note that this may be nuanced with other contextual restrictions, in which case, it may be preferable to read from servingRestrictions directly. Can be used to filter the response of the creatives.list method.

Note: This field is used in responses only. Any value specified here in a request is ignored.
servingRestrictions[]

object(ServingRestriction)

The granular status of this ad in specific contexts. A context here relates to where something ultimately serves (for example, a physical location, a platform, an HTTPS vs HTTP request, or the type of auction).

Note: This field is used in responses only. Any value specified here in a request is ignored.
detectedAdvertiserIds[]

string (int64 format)

Detected advertiser IDs, if any.

Note: This field is used in responses only. Any value specified here in a request is ignored.
detectedProductCategories[]

number

Detected product categories, if any. See the ad-product-categories.txt file in the technical documentation for a list of IDs.

Note: This field is used in responses only. Any value specified here in a request is ignored.
detectedLanguages[]

string

The detected languages for this creative. The order is arbitrary. The codes are 2 or 5 characters and are documented at https://developers.google.com/adwords/api/docs/appendix/languagecodes.

Note: This field is used in responses only. Any value specified here in a request is ignored.
detectedDomains[]

string

The detected domains for this creative.

Note: This field is used in responses only. Any value specified here in a request is ignored.
detectedSensitiveCategories[]

number

Detected sensitive categories, if any. See the ad-sensitive-categories.txt file in the technical documentation for a list of IDs. You should use these IDs along with the excluded-sensitive-category field in the bid request to filter your bids.

Note: This field is used in responses only. Any value specified here in a request is ignored.
corrections[]
(deprecated)

object(Correction)

This item is deprecated!

Shows any corrections that were applied to this creative.

Note: This field is used in responses only. Any value specified here in a request is ignored.
filteringStats

object(FilteringStats)

The filtering stats for this creative.

Note: This field is used in responses only. Any value specified here in a request is ignored.
version

number

The version of this creative.

Note: This field is used in responses only. Any value specified here in a request is ignored.
apiUpdateTime

string (Timestamp format)

The last update timestamp of the creative via API.

A timestamp in RFC3339 UTC "Zulu" format, accurate to nanoseconds. Example: "2014-10-02T15:01:23.045123456Z".

Note: This field is used in responses only. Any value specified here in a request is ignored.
Union field content. Creative content in different formats. content can be only one of the following:
html

object(HtmlContent)

An HTML creative.

object(VideoContent)

A video creative.

native

object(NativeContent)

A native creative.

ServingRestriction

A representation of the status of an ad in a specific context. A context here relates to where something ultimately serves (for example, a user or publisher geo, a platform, an HTTPS vs HTTP request, or the type of auction).

Note: This field is used in responses only. Any value specified here in a request is ignored.

    }
  ],

status

enum(ServingRestrictionStatus)

The status of the creative in this context (for example, it has been explicitly disapproved or is pending review).

contexts[]

object(ServingContext)

The contexts for the restriction.

disapprovalReasons[]

object(Disapproval)

Any disapprovals bound to this restriction. Only present if status=DISAPPROVED. Can be used to filter the response of the creatives.list method.

Union field context. The type of context for the serving restriction. context can be only one of the following:
all

enum(SimpleContext)

Matches all contexts.

location

object(LocationContext)

Matches impressions coming from users or publishers in a specific location.

platform

object(PlatformContext)

Matches impressions coming from a particular platform.

auctionType

object(AuctionContext)

Matches impressions for a particular auction type.

securityType

object(SecurityContext)

Matches impressions for a particular security type.

appType

object(AppContext)

Matches impressions for a particular app type.

SimpleContext

A simple context is one which does not have additional information associated with it. It is used for the All context. See also, LocationContext, PlatformContext, AuctionContext, and SecurityContext for contrast.

Note: This field is used in responses only. Any value specified here in a request is ignored.

Enums
SIMPLE_CONTEXT A simple context.

Fields
date

object(Date)

The day during which the data was collected. The data is collected from 00:00:00 to 23:59:59 PT. During switches from PST to PDT and back, the day may contain 23 or 25 hours of data instead of the usual 24.
