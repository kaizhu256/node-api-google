Ad Exchange APIs and Protocol Terms of Service

Last modified: September 3, 2015

By using this API, you consent to be bound by these terms in addition to the Google APIs Terms of Service ("API ToS") at https://developers.google.com/terms.

Section 1: Security

You must use reasonable efforts to keep all Ad Exchange API data in a secure environment at all times according to commonly acceptable security standards for enterprise data, but in an environment at least as secure as that for your enterprise data. All transfer of CPM data using the Ad Exchange Real-Time Bidding Protocol must be secured using the Google-provided encryption library and implemented in accordance with instructions provided by Google.

Section 2: For Marketplace API

You are permitted to display in your user interface, data accessed through the Marketplace API provided such data is displayed for the purpose of enabling Marketplace transactions, and is accurately and completely represented as relates to the deal or inventory opportunity. Such data may not be sub-licensed, re-sold, or otherwise shared with any other party.
