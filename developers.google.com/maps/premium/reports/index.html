Premium Plan Reporting Overview

Reporting tools

This page is only for customers with the new version of the Google Maps APIs Premium Plan, which became available in January 2016.

Have a previous Maps APIs for Work or Maps API for Business license? See our Maps APIs for Work Licenses guide. To determine whether you have a previous license: In the Google Cloud Support Portal, click Maps: Usage Report on the left. If the ID at the top of the report is in the following format, you have the new Premium Plan:
gme-[company] & proj-[number] ([type])
Otherwise, you have a previous license.

With the Google Maps APIs Premium Plan, you have access to several reporting tools that provide different types of usage data.

Support Portal usage reports

The Google Maps Usage Reports tool in the Google Cloud Support Portal provides:

  • Usage data for any API requests that use either a client ID or API key
  • Total daily and monthly Maps APIs usage totals for each API, with a 1-day delay
  • Total daily and monthly Maps APIs Credits usage for each API, with a 3-day delay
  • Error reporting for web service requests that use a client ID
  • Channel reports for the Google Maps JavaScript API, image APIs (Google Static Maps API and Google Street View Image API), and web service APIs

Support Portal Analytics

The Google Maps Analytics tool in the Google Cloud Support Portal provides real-time Google Maps Analytics data for Google Maps JavaScript API requests that use a client ID. With Analytics, you can see details about how users are interacting with your application, as well as from which countries they are requesting services that your application provides, such as geocoding.

Google API Console reports

The Google API Console reports provide real-time usage data for any APIs for which you use an API key. Note that the reports provide data for up to 30 days only.

To view Google API Console reports:

  1. Sign in to the Google API Console using your Google Account.
  2. In the project drop-down menu, select the project created for you when you purchased the Premium Plan.
  3. Select a Google Maps API, and then click Usage.
